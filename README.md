# Proyecto Final DataOps - Docker y Jenkins

Este repositorio contiene un flujo DataOps con CI/CD usando Jenkins, Docker y Python.

El pipeline realiza lo siguiente:

1. Descarga el código desde GitHub.
2. Construye una imagen Docker.
3. Ejecuta un contenedor Python.
4. Genera un archivo Excel de comisiones.
5. Valida la existencia del artefacto generado.

Herramientas usadas:

- Azure VM Ubuntu
- Docker
- Jenkins
- GitHub
- Python
- Pandas
- OpenPyXL
